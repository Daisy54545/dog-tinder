# 🐶 TinDog - Dog Tinder (Frontend)

A responsive frontend web application inspired by Tinder, designed for dog owners to connect their pets with others nearby.

## 🚀 Features
- Clean and modern landing page design  
- Sections for features, testimonials, and pricing  
- Responsive layout for different screen sizes  
- Interactive UI elements and buttons  

## 🛠️ Tech Stack
- HTML  
- CSS  
- Bootstrap
